= Diagrams Parity
:doctype: article

A Mermaid flowchart:

[mermaid]
....
graph LR
  A[Start] --> B{Choice}
  B -->|yes| C[Do the thing]
  B -->|no| D[Skip it]
....

A Graphviz directed graph:

[graphviz]
....
digraph {
  rankdir=LR;
  a -> b -> c;
  a -> c;
}
....
